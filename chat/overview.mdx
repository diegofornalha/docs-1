O Cursor Chat é uma ferramenta poderosa que permite que você faça perguntas ou resolva problemas na sua base de código com os modelos de linguagem mais avançados, tudo dentro do seu editor. Ele é projetado para integrar-se perfeitamente ao seu fluxo de trabalho, fornecendo contexto relevante e sugestões precisas.

## Como Funciona

O Cursor Chat utiliza modelos de linguagem para fornecer respostas informadas e contextuais. Para que as respostas sejam precisas, o Cursor Chat inclui automaticamente o contexto relevante da sua base de código, pesquisa na web, e referências especificadas pelo usuário.

### Recursos Principais

- **Contexto Automático**: Inclui automaticamente o arquivo atual e outros contextos relevantes.
- **Pesquisa na Web**: Usa `@Web` para buscar informações atualizadas e relevantes.
- **Histórico de Chat**: Salva todas as conversas para fácil referência futura.

### Exemplos de Uso

- **Resolução de Problemas**: Use o Cursor Chat para diagnosticar e resolver problemas complexos no código, aproveitando o contexto completo da base de código.
- **Geração de Código**: Peça ao Cursor Chat para gerar trechos de código com base em descrições ou requisitos específicos.
- **Revisão de Código**: Utilize o Cursor Chat para revisar alterações de código, garantindo que todas as dependências e interações sejam consideradas.

### Integração com Outros Recursos

- **Revisão de IA**: Combine o Cursor Chat com a Revisão de IA para uma análise mais profunda e abrangente do código.
- **Símbolos @**: Use símbolos @ para adicionar contexto adicional, como arquivos, pastas ou links, ao chat.

### Dicas para Melhor Uso

- **Forneça Contexto Completo**: Sempre que possível, inclua o máximo de contexto relevante para garantir que a IA forneça respostas precisas.
- **Use o Histórico de Chat**: Revise conversas anteriores para insights ou soluções que possam ser aplicáveis a problemas atuais.
- **Combine com Outros Recursos**: Use o Cursor Chat em conjunto com outros recursos do Cursor para maximizar a eficiência e a qualidade do trabalho.

### Estratégia de Uso

- **Especificidade**: Seja específico ao formular suas perguntas ou solicitações para garantir que a IA entenda suas necessidades.
- **Colaboração**: Compartilhe insights ou soluções do Cursor Chat com sua equipe para promover a colaboração e a inovação.

### Mensagens do Usuário e da IA

As mensagens do usuário contêm o texto que você digita, junto com o contexto que você referenciou. Você pode voltar a qualquer mensagem anterior do usuário para editar e executar novamente suas consultas. Isso irá sobrescrever quaisquer mensagens após isso e gerar novas.

As mensagens da IA são as respostas geradas a partir do modelo de IA que você escolheu. Elas são emparelhadas com a mensagem do usuário anterior. As mensagens da IA podem conter blocos de código analisados que podem ser adicionados à sua base de código com [aplicação instantânea](/chat/apply).

### Histórico de Chat

Ao pressionar o botão "Chats Anteriores" no canto superior direito do painel de IA, ou pressionando `Ctrl/⌘ + Alt/Option + L`, você pode ver o histórico de chat. Você pode clicar em qualquer thread de chat para voltar e ver as mensagens que compõem esse thread, e você também pode modificar o título do thread clicando no ícone de caneta, ou excluir o thread clicando no ícone de lixeira ao passar o mouse sobre o thread no histórico.

O título de um thread do Cursor é apenas as primeiras palavras da primeira mensagem do usuário.

### Contexto Padrão

Por padrão, o Cursor Chat inclui o arquivo atual como contexto. Você pode enviar uma consulta sem incluir nenhum contexto pressionando `Alt/Option Enter` ao enviar, ou ativar `Padrão para nenhum contexto` em `Configurações do Cursor` > `Recursos` > `Chat`. Chats que começam com uma solicitação sem contexto não adicionarão contexto para nenhuma mensagem.

Conforme você digita, você pode ver o que será incluído no contexto nas pílulas abaixo da caixa de entrada.

### Adicionando Contexto

Por padrão, as mensagens do usuário conterão o texto que você digita, junto com o contexto que você referenciou. Você pode adicionar mais contexto personalizado a cada bolha com símbolos @, e por padrão, o arquivo visualizado atualmente também será usado como contexto na mensagem do usuário.

Veja as páginas de [símbolos @](/context/@-symbols/@-files) para mais informações.

### Correção de IA no Chat

Um recurso conveniente para corrigir erros de lint na sua base de código é usar a correção de IA no chat. Para fazer isso, passe o mouse sobre o erro no editor e clique no botão azul de correção de IA que aparece.

O atalho de teclado para isso seria `Ctrl/⌘ + Shift + E`.

### Chat de Contexto Longo (Beta)

Você pode habilitar o chat de contexto longo indo para `Configurações do Cursor` > `Beta` > `Chat de Contexto Longo`. Com isso habilitado, você pode alternar entre diferentes modos de chat através de `Ctrl/⌘ .`.

O chat de contexto longo permite que você inclua pastas inteiras como contexto, já que os modelos suportados têm uma janela de contexto maior. Confira mais sobre os modelos apenas para contexto longo [aqui](/advanced/models#long-context-only-models).

Com essas melhorias, o arquivo `overview.mdx` fornece uma visão mais clara e abrangente sobre como usar o Cursor Chat de forma eficaz, ajudando a melhorar a produtividade e a qualidade do trabalho com contexto integrado e recursos avançados.

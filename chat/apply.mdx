O `Aplicar` do Cursor permite que você integre rapidamente uma sugestão de bloco de código do chat em seu código.

### Aplicar Blocos de Código

Para aplicar uma sugestão de bloco de código, você pode pressionar o botão de play no canto superior direito de cada bloco de código do chat.

<Frame>
  <img src="/images/chat/apply.png" />
</Frame>

Isso editará seu arquivo para incorporar o código produzido pelo Chat. Como você pode adicionar mais contexto e ter mais interação com o modelo no Chat,
recomendamos Chat + Aplicar para mudanças de código mais complexas impulsionadas por IA.

### Aceitar ou Rejeitar

Depois de aplicar um bloco de código, você pode revisar os diffs e aceitar ou rejeitar as alterações. Você também pode clicar
nos botões "Aceitar" ou "Rejeitar" no canto superior direito do bloco de código do chat.

`Ctrl/⌘ Enter` para aceitar, `Ctrl/⌘ Backspace` para rejeitar.

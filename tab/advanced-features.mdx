### Recursos Avançados do Cursor

Os recursos avançados do Cursor são projetados para aumentar a produtividade e a eficiência dos desenvolvedores, oferecendo funcionalidades que vão além do autocompletar básico. Esses recursos permitem edições mais complexas, navegação aprimorada e integração com ferramentas de IA.

</Frame>

## Tab em Peek

O recurso Tab em Peek permite que você use o Cursor Tab nas visualizações "Ir para Definição" ou "Ir para Definição de Tipo". Isso é útil, por exemplo, ao adicionar um novo argumento a uma chamada de função, permitindo que você modifique rapidamente a definição e todos os seus usos.

### Exemplos de Uso

- **Refatoração de Código**: Use Tab em Peek para refatorar funções ou classes, garantindo que todas as referências sejam atualizadas de forma consistente.
- **Correção de Erros**: Identifique e corrija rapidamente erros em definições de funções ou tipos, sem precisar navegar manualmente por todo o código.

## Previsão do Cursor

O Cursor pode prever para onde você irá após uma edição aceita. Se disponível, você poderá pressionar tab para ir para o próximo local, permitindo que você navegue rapidamente pelas edições.

### Exemplos de Uso

- **Edição Sequencial**: Ao fazer alterações em série, use a Previsão do Cursor para navegar rapidamente entre os locais de edição.
- **Fluxo de Trabalho Eficiente**: Mantenha um fluxo de trabalho contínuo e eficiente, minimizando interrupções ao alternar entre edições.

## Aceitações Parciais

Você pode aceitar a próxima palavra de uma sugestão pressionando `Ctrl/⌘` e a seta para a direita (ou configurando `editor.action.inlineSuggest.acceptNextWord` para o atalho de teclado de sua preferência). Isso permite que você integre sugestões de forma mais granular, ajustando-as conforme necessário.

### Exemplos de Uso

- **Personalização de Código**: Aceite partes de uma sugestão para personalizar o código de acordo com suas necessidades específicas.
- **Correção de Sintaxe**: Use aceitações parciais para corrigir rapidamente erros de sintaxe ou completar expressões.

## Integração com Outros Recursos

Para maximizar a eficácia dos recursos avançados, considere integrá-los com outros recursos do Cursor, como:

- **Revisão de IA**: Use recursos avançados em conjunto com a Revisão de IA para garantir que as alterações de código sejam otimizadas para qualidade e desempenho.
- **Símbolos @**: Combine recursos avançados com símbolos @ para fornecer um contexto ainda mais rico e específico para a IA.

## Dicas para Melhor Uso

- **Explorar Funcionalidades**: Experimente diferentes combinações de recursos avançados para encontrar o fluxo de trabalho que melhor se adapta às suas necessidades.
- **Configurações Personalizadas**: Ajuste as configurações do Cursor para otimizar o uso de recursos avançados, garantindo que eles funcionem de acordo com suas preferências.
- **Colaboração**: Compartilhe dicas e truques sobre o uso de recursos avançados com sua equipe para melhorar a colaboração e a eficiência.

## Estratégia de Uso

- **Especificidade**: Seja específico ao usar recursos avançados para garantir que a IA forneça as alterações mais relevantes.
- **Iteração Rápida**: Use o feedback da IA para iterar rapidamente sobre suas alterações, garantindo que o código esteja sempre em seu melhor estado.

Com essas melhorias, o arquivo `advanced-features.mdx` fornece uma visão mais clara e abrangente sobre como usar os recursos avançados do Cursor de forma eficaz, ajudando a melhorar a produtividade e a qualidade do trabalho com ferramentas de edição e navegação aprimoradas.

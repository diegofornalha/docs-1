### Tab em Peek

Você também pode usar o Cursor Tab nas visualizações "Ir para Definição" ou "Ir para Definição de Tipo". Isso é útil, por exemplo, ao adicionar um novo argumento a uma chamada de função.

<Frame>
  <img src="/images/cpp/cpp-in-peek.png" />
</Frame>

Nós especialmente gostamos de usar isso no vim em conjunto com `gd` para, por exemplo, modificar uma definição de função e depois corrigir todos os seus usos de uma só vez.

### <div className="flex items-center" style={{ gap: '6px' }}> <span className="cursor-pointer my-0">Previsão do Cursor</span></div>

O Cursor também pode prever para onde você irá após uma edição aceita.
Se disponível, você poderá pressionar tab para ir para o próximo local, permitindo que você navegue rapidamente pelas edições.

### <div className="flex items-center" style={{ gap: '6px' }}> <span className="cursor-pointer my-0">Aceitações Parciais</span></div>

Você pode aceitar a próxima palavra de uma sugestão pressionando `Ctrl/⌘` e a seta para a direita (ou configurando `editor.action.inlineSuggest.acceptNextWord` para o atalho de teclado de sua preferência).

Para habilitar aceitações parciais, navegue até `Configurações do Cursor` > `Recursos` > `Cursor Tab`.

### Melhorias do Tab

A maior diferença é a forma como o Cursor e o GitHub Copilot completam o código.

O GitHub Copilot pode inserir texto na posição do seu cursor. Ele não pode editar o código ao redor do seu cursor ou remover texto.

O Cursor pode inserir texto no seu cursor, e muito mais:

- Edições de múltiplos caracteres

  </Frame>
- Edições baseadas em instruções

Além disso, o Cursor tem um histórico de suas alterações recentes na janela de contexto, então ele sabe o que você está tentando fazer a seguir.

### Migrar do GitHub Copilot

Como o Cursor vem por padrão com o GitHub Copilot, você pode ter o GitHub Copilot e o Cursor instalados ao mesmo tempo. Recomendamos desativar o GitHub Copilot quando você quiser usar o Cursor.

Por padrão, o Cursor tem precedência sobre o GitHub Copilot. Se você quiser usar o GitHub Copilot, pode [desativar o Cursor](/tab/overview#copilot-settings) nas configurações.

---
title: Ignorar Arquivos
---

Para ignorar arquivos a serem incluídos em recursos do Cursor, como [indexação da base de código](/context/codebase-indexing), você
pode usar um arquivo `.cursorignore` na raiz do seu projeto. Ele funciona da mesma forma que o `.gitignore` funciona para o git.

O `.cursorignore` respeita o `.gitignore`. Se você já tiver um `.gitignore`, os arquivos serão ignorados por padrão.
Se você quiser ignorar arquivos adicionais, pode adicioná-los ao arquivo `.cursorignore`.

### Um Exemplo de Arquivo `.cursorignore`

```
# Ignorar todos os arquivos no diretório `dist`
dist/

# Ignorar todos os arquivos `.log`
*.log

# Ignorar arquivo específico `config.json`
config.json
```

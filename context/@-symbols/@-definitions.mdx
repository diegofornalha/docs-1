### `@Definitions`

O recurso `@Definitions` permite que você adicione todas as definições próximas ao Cmd K como contexto. Isso é especialmente útil quando você está trabalhando em um projeto grande e precisa de uma visão rápida das definições de funções, classes ou variáveis próximas ao seu ponto de edição.

<Frame>
  <img src="/images/context/@definitions.png" />
</Frame>

## Como Usar

Para usar `@Definitions`, basta digitar `@` seguido por `Definitions` no Cmd K. O Cursor irá automaticamente incluir todas as definições relevantes próximas ao seu ponto de edição como parte do contexto para a IA.

### Exemplos de Uso

- **Refatoração de Código**: Use `@Definitions` para garantir que todas as definições relevantes sejam consideradas ao refatorar uma função ou classe.
- **Depuração**: Inclua definições próximas ao depurar problemas complexos para que a IA possa entender o contexto completo.
- **Geração de Documentação**: Referencie definições ao gerar documentação para garantir que todos os detalhes relevantes sejam incluídos.

### Integração com Outros Recursos

Para uma análise mais abrangente, considere usar `@Definitions` em conjunto com outros recursos do Cursor, como:

- **Revisão de IA**: Para garantir que a IA tenha acesso a todas as definições relevantes durante a revisão de código.
- **Indexação da Base de Código**: Para melhorar a precisão das respostas da IA ao usar definições como contexto.

### Dicas para Melhor Uso

- **Verifique as Definições**: Sempre verifique as definições incluídas para garantir que você está referenciando as partes corretas do código.
- **Combine com Símbolos @**: Use `@Definitions` em conjunto com outros símbolos @ para fornecer um contexto ainda mais rico para a IA.
- **Contexto Relevante**: Limite o uso de `@Definitions` a definições que são diretamente relevantes para a consulta, para evitar sobrecarregar a IA com informações desnecessárias.

### Estratégia de Uso

- **Especificidade**: Seja específico ao referenciar definições para garantir que a IA receba o contexto mais relevante.
- **Colaboração**: Use `@Definitions` para colaborar com outros desenvolvedores, fornecendo contexto detalhado sobre alterações ou problemas específicos.

Com essas melhorias, o arquivo `@-definitions.mdx` fornece uma visão mais clara e abrangente sobre como usar o recurso `@Definitions` de forma eficaz no Cursor, ajudando a melhorar a produtividade e a qualidade do código.

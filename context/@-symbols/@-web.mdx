### `@Web`

<Frame>
  <img src="/images/context/@web.png" />
</Frame>

Com `@Web`, o Cursor constrói uma consulta de busca com base na consulta e no contexto que você forneceu, e pesquisa na web para encontrar informações relevantes como contexto adicional. Isso é particularmente útil para encontrar as informações mais atualizadas.

### Sempre Ativo

Você pode fazer o Cursor buscar na web para o chat em cada consulta ativando "Sempre buscar na web" em `Configurações do Cursor` > `Recursos` > `Chat`. Isso é equivalente a usar `@web` em cada consulta.

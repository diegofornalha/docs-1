### `@Code`

Para referenciar seções específicas de código, você pode usar o símbolo `@Code`.

### Pré-visualização de Código

Semelhante ao símbolo [`@Files`](/context/@-symbols/@-files), o Cursor mostrará uma pré-visualização do conteúdo do código para que você possa verificar se o código que está referenciando é o correto.

</Frame>

### Do Editor

Outra maneira de adicionar trechos de código como contexto é selecionar o código que você deseja referenciar e clicar em "Adicionar ao Chat" (`Ctrl/⌘ Shift L`) ou "Adicionar à Edição" (`Ctrl/⌘ Shift K`).

Isso adicionará o trecho de código selecionado à caixa de entrada do Chat ou à barra de prompt Cmd K atualmente ativa.

<Frame>
  <img src="/images/context/@code-select.png" />
</Frame>

Para adicionar um código selecionado a um novo chat, você pode pressionar `Ctrl/⌘ L`.

### `@Files`

Nas caixas de entrada de IA, como no Cursor Chat e Cmd K, você pode referenciar arquivos inteiros usando `@Files`.
Além disso, se você continuar digitando após `@`, verá os resultados da busca de arquivos após a estratégia [`@Code`](/context/@-symbols/@-code).

Para garantir que o arquivo que você está referenciando é o correto, o Cursor mostrará uma pré-visualização do caminho do arquivo. Isso é especialmente útil quando você tem vários arquivos com o mesmo nome em pastas diferentes.

### Referências de Arquivos Longos no Chat

No Chat do Cursor, se o conteúdo de um arquivo for muito longo, o Cursor dividirá o arquivo em partes menores e as reclassificará com base na relevância para a consulta.

### Estratégia de Fragmentação Cmd K

Para Cmd K, o Cursor usa as referências de arquivo de maneira diferente com base no comprimento do conteúdo.

- auto
  - Escolhe automaticamente uma das três estratégias de leitura com base no tamanho do arquivo
- arquivo completo
  - O arquivo inteiro é usado como contexto.
- esboço
  - O Cursor analisa o esboço do arquivo e usa as informações como contexto.
- fragmentos
  - O Cursor divide o arquivo em partes menores e escolhe a mais relevante.

### Arrastar e Soltar

Você pode arrastar e soltar arquivos da barra lateral principal no Chat ou Cmd K para adicioná-los como contexto também.

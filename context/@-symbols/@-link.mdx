### `@https://seu-link.com`

O recurso `@Link` permite que o Cursor visite um link antes de responder, adicionando o conteúdo do link como contexto para a IA. Isso é especialmente útil quando você precisa que a IA considere informações externas ao seu projeto atual.

</Frame>

## Como Usar

Para usar `@Link`, basta digitar `@` seguido pelo URL que você deseja referenciar. O Cursor irá automaticamente acessar o link e incluir o conteúdo relevante como parte do contexto para a IA.

### Exemplos de Uso

- **Referência de Documentação**: Use `@Link` para incluir documentação externa relevante ao seu projeto, permitindo que a IA forneça respostas mais informadas.
- **Artigos Técnicos**: Inclua links para artigos técnicos ou blogs que discutem conceitos ou tecnologias que você está utilizando.
- **Recursos de Aprendizado**: Adicione links para tutoriais ou cursos online que podem ajudar a IA a entender melhor o contexto do seu projeto.

### Remover Links

Por padrão, o Cursor automaticamente analisa links e os transforma em `@Links` no Cursor Chat. Se preferir ter o link como texto simples, clique no link e depois clique em `Desvincular`.

### Integração com Outros Recursos

- **Revisão de IA**: Use `@Link` em conjunto com a Revisão de IA para garantir que a IA tenha acesso a todas as informações relevantes ao revisar seu código.
- **Símbolos @**: Combine `@Link` com outros símbolos @ para fornecer um contexto ainda mais rico e específico para a IA.

### Dicas para Melhor Uso

- **Verifique a Relevância do Link**: Certifique-se de que os links que você está usando são relevantes e atualizados para garantir que a IA forneça informações precisas.
- **Organização de Links**: Mantenha uma lista organizada de links úteis que você pode usar regularmente em seus projetos.
- **Contexto Relevante**: Use `@Link` para incluir apenas links que são diretamente relevantes para sua consulta, evitando sobrecarregar a IA com informações desnecessárias.

### Estratégia de Uso

- **Especificidade**: Seja específico ao escolher links para garantir que a IA receba o contexto mais relevante.
- **Colaboração**: Use `@Link` para colaborar com outros desenvolvedores, fornecendo contexto detalhado sobre recursos externos ou referências.

Com essas melhorias, o arquivo `@-link.mdx` fornece uma visão mais clara e abrangente sobre como usar o recurso `@Link` de forma eficaz no Cursor, ajudando a melhorar a produtividade e a qualidade do trabalho com informações externas integradas.

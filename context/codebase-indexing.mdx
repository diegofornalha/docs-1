### Indexar sua Base de Código

Para obter respostas mais precisas e melhores sobre a base de código usando `@codebase` ou `Ctrl/⌘ Enter`, você pode indexar sua base de código. Nos bastidores, o Cursor
calcula embeddings para cada arquivo na sua base de código e os usará para melhorar a precisão das respostas da sua base de código.

Seu índice de base de código será automaticamente sincronizado com suas alterações mais recentes na base de código.

O status da indexação da sua base de código está em `Configurações do Cursor` > `Recursos` > `Indexação da Base de Código`.

### Configurações Avançadas

Por padrão, o Cursor indexará todos os arquivos na sua base de código.

Você também pode expandir a seção `Mostrar Configurações` para acessar mais opções avançadas.
Aqui, você pode decidir se deseja habilitar a indexação automática para novos repositórios e configurar os arquivos
que o Cursor ignorará durante a indexação do repositório, além das configurações do seu .gitignore.

Se você tiver arquivos de conteúdo grandes em seu projeto que a IA definitivamente não precisa ler, [ignorar esses arquivos](/context/ignore-files) pode melhorar a precisão das respostas.

Cursor é um fork do VS Code, projetado para integrar funcionalidades de IA que otimizam o fluxo de trabalho dos desenvolvedores. Isso nos permite focar em criar a melhor maneira de programar com IA, enquanto oferecemos uma experiência de edição de texto familiar.

## Por Que Migrar para o Cursor?

Migrar para o Cursor oferece várias vantagens em relação ao uso do VS Code padrão:

- **Integração de IA**: O Cursor oferece integração nativa com IA, permitindo sugestões de código mais inteligentes e revisões automáticas.
- **Funcionalidades Avançadas**: Recursos como Revisão de IA, símbolos @, e indexação da base de código proporcionam um ambiente de desenvolvimento mais eficiente.
- **Experiência Familiar**: Como um fork do VS Code, o Cursor mantém a interface e a experiência de usuário que você já conhece, facilitando a transição.

### Importar Extensões, Temas, Configurações e Atalhos

Você pode importar sua configuração do VS Code para o Cursor com um clique. Navegue até `Configurações do Cursor` > `Geral` > `Conta`. Isso inclui:

- **Extensões**: Todas as suas extensões do VS Code podem ser importadas, garantindo que você mantenha suas ferramentas favoritas.
- **Temas**: Personalize o Cursor com seus temas preferidos do VS Code.
- **Configurações**: Todas as suas configurações personalizadas serão transferidas, minimizando a necessidade de ajustes manuais.
- **Atalhos de Teclado**: Mantenha seus atalhos de teclado personalizados para uma transição suave.

### Mantendo-se Atualizado

Periodicamente, rebaseamos o Cursor na versão mais recente do VS Code. Isso garante que você tenha acesso às últimas melhorias e correções de bugs, além das funcionalidades exclusivas do Cursor.

### Por que não uma Extensão?

Como um aplicativo independente, o Cursor tem mais controle sobre a interface do editor, permitindo uma maior integração com IA. Alguns de nossos recursos, como Cursor Tab e CMD-K, não são possíveis como plugins em ambientes de codificação existentes.

### Configurações

Você pode abrir o painel de configurações específicas do Cursor clicando no botão de engrenagem no canto superior direito, pressionando `Ctrl/⌘ + Shift + J`, ou com `Ctrl/⌘ + Shift + P` e digitando `Configurações do Cursor`.

Você pode abrir as configurações específicas do VS Code com `Ctrl/⌘ + Shift + P`, e então digitando `Configurações do VS Code`.

### Por que a Barra de Atividades no Cursor é horizontal?

</Frame>

A barra de atividades é horizontal por padrão para economizar espaço para o chat. Se você preferir a barra de atividades vertical normal, pode ir nas configurações do VS Code e definir `workbench.activityBar.orientation` para `vertical`, e reiniciar o Cursor.

### Dicas para uma Migração Suave

- **Explore os Recursos do Cursor**: Aproveite o tempo para explorar os recursos exclusivos do Cursor, como Revisão de IA e símbolos @, para maximizar sua produtividade.
- **Personalize seu Ambiente**: Use as opções de importação para personalizar o Cursor de acordo com suas preferências, garantindo uma experiência de usuário otimizada.
- **Aproveite o Suporte**: Se encontrar dificuldades durante a migração, utilize os recursos de suporte do Cursor, como documentação e fóruns, para obter ajuda.

Com essas melhorias, o arquivo `migrate-from-vscode.mdx` fornece uma visão mais clara e abrangente sobre como migrar do VS Code para o Cursor, destacando as vantagens e fornecendo dicas práticas para facilitar a transição.

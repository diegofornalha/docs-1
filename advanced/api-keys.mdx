### Chaves de API da OpenAI

O Cursor permite que você insira sua própria chave de API da OpenAI para enviar quantas mensagens de IA quiser, por sua própria conta.

Você pode obter sua própria chave de API [aqui](https://platform.openai.com/account/api-keys). Para usar sua própria chave de API, vá para `Configurações do Cursor` > `Modelos` > `Chave de API da OpenAI` e insira sua chave de API. Em seguida, clique no botão "Verificar". Uma vez que sua chave for validada, sua chave de API da OpenAI será habilitada.

### Chaves de API do Google

Para as chaves de API do Google, você pode definir sua própria chave de API para que você use modelos do Google, como `gemini-1.5-flash-500k`, por sua própria conta.

### Minha chave de API será armazenada ou sairá do meu dispositivo?

Sua chave de API não será armazenada, mas será enviada para o nosso servidor com cada solicitação. Todas as solicitações são roteadas através do nosso backend, porque é lá que fazemos a construção final do prompt.

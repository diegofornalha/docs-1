title: Revisão de IA (Beta)

A Revisão de IA é um recurso poderoso que permite revisar suas alterações recentes na base de código para identificar possíveis bugs e melhorar a qualidade do código. Este recurso é especialmente útil para desenvolvedores que buscam otimizar seu fluxo de trabalho e garantir a estabilidade do código.

## Como Funciona

A Revisão de IA analisa suas alterações recentes e fornece feedback sobre possíveis problemas. Você pode clicar em itens de revisão individuais para ver o contexto completo no editor e conversar com a IA para obter mais informações.

### Instruções Personalizadas de Revisão

Para que a Revisão de IA funcione a seu favor, você pode fornecer instruções personalizadas para a IA se concentrar. Por exemplo, se você quiser que a IA se concentre em questões relacionadas ao desempenho, você poderia colocar:

```
focar no desempenho do meu código
```

Dessa forma, a Revisão de IA se concentrará no desempenho do seu código ao examinar suas alterações.

#### Exemplos de Instruções

- **Segurança**: "Identificar vulnerabilidades de segurança no meu código."
- **Legibilidade**: "Melhorar a legibilidade e clareza do código."
- **Manutenibilidade**: "Sugerir melhorias para facilitar a manutenção futura."

### Opções de Revisão

Atualmente, você tem várias opções para escolher na revisão:

- `Revisar Estado de Trabalho`
  - Isso revisará suas alterações não comitadas.
- `Revisar Diferença com a Ramificação Principal`
  - Isso revisará a diferença entre seu estado de trabalho atual e a ramificação principal.
- `Revisar Último Commit`
  - Isso revisará o último commit que você fez.

### Integração com Outros Recursos

Para uma análise mais abrangente, considere usar a Revisão de IA em conjunto com outros recursos do Cursor, como:

- **Indexação da Base de Código**: Para garantir que a IA tenha acesso a todo o contexto necessário.
- **Símbolos @**: Para referenciar partes específicas do código ou documentação durante a revisão.

### Dicas para Melhor Uso

- **Contexto Completo**: Sempre forneça o máximo de contexto possível para a IA, usando símbolos @ para incluir arquivos ou trechos de código relevantes.
- **Feedback Iterativo**: Use o feedback da IA para iterar rapidamente sobre suas alterações, garantindo que o código esteja sempre em seu melhor estado.

Com essas melhorias, a Revisão de IA pode se tornar uma parte essencial do seu fluxo de trabalho de desenvolvimento, ajudando a manter a qualidade e a eficiência do seu código.

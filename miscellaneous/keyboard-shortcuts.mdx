### Atalhos de Teclado do Cursor

Os atalhos de teclado são uma maneira eficaz de aumentar a produtividade ao usar o Cursor, permitindo que você execute ações rapidamente sem precisar navegar por menus. Este documento fornece uma visão geral dos atalhos de teclado disponíveis e como você pode personalizá-los para atender às suas necessidades.

## Atalhos Padrão

Aqui estão alguns dos atalhos de teclado mais comuns no Cursor:

- **Abrir Configurações**: `Ctrl/⌘ + ,`
- **Abrir Painel de Comandos**: `Ctrl/⌘ + Shift + P`
- **Alternar Painel de IA**: `Ctrl/⌘ + L`
- **Aceitar Sugestão**: `Tab`
- **Rejeitar Sugestão**: `Esc`
- **Aceitar Próxima Palavra**: `Ctrl/⌘ + →`
- **Aplicar Código**: `Ctrl/⌘ + Enter`
- **Rejeitar Código Aplicado**: `Ctrl/⌘ + Backspace`

## Personalização de Atalhos

Você pode personalizar os atalhos de teclado no Cursor para melhor se adequar ao seu fluxo de trabalho. Para fazer isso, siga estas etapas:

1. **Acesse as Configurações**: Abra o painel de comandos com `Ctrl/⌘ + Shift + P` e digite `Configurações do Cursor`.
2. **Navegue até Atalhos de Teclado**: Procure por "Atalhos de Teclado" na barra de pesquisa de configurações.
3. **Personalize os Atalhos**: Clique no atalho que deseja alterar e insira a nova combinação de teclas.

## Exemplos de Uso

- **Navegação Rápida**: Use `Ctrl/⌘ + L` para alternar rapidamente para o painel de IA e focar no chat, economizando tempo ao buscar assistência da IA.
- **Edição Eficiente**: Aceite sugestões de código com `Tab` para integrar rapidamente alterações sugeridas pela IA.
- **Gerenciamento de Código**: Use `Ctrl/⌘ + Enter` para aplicar blocos de código sugeridos, garantindo que as alterações sejam implementadas de forma eficiente.

## Dicas para Melhor Uso

- **Memorize Atalhos Comuns**: Familiarize-se com os atalhos que você usa com mais frequência para aumentar sua eficiência.
- **Personalize para Suas Necessidades**: Ajuste os atalhos de teclado para se alinhar com seu estilo de trabalho e preferências pessoais.
- **Explore Novos Atalhos**: Experimente novos atalhos para descobrir maneiras adicionais de otimizar seu fluxo de trabalho.

## Integração com Outros Recursos

- **Revisão de IA**: Use atalhos de teclado em conjunto com a Revisão de IA para navegar rapidamente entre sugestões e aplicar alterações.
- **Símbolos @**: Combine atalhos de teclado com símbolos @ para fornecer contexto adicional ao interagir com a IA.

### Estratégia de Uso

- **Especificidade**: Seja específico ao configurar atalhos para garantir que eles atendam às suas necessidades de fluxo de trabalho.
- **Colaboração**: Compartilhe suas configurações de atalhos de teclado com sua equipe para melhorar a colaboração e a eficiência.

Com essas melhorias, o arquivo `keyboard-shortcuts.mdx` fornece uma visão mais clara e abrangente sobre como usar e personalizar atalhos de teclado no Cursor, ajudando a melhorar a produtividade e a qualidade do trabalho com navegação e edição otimizadas.
